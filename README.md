## Tech Stack

- Pure HTML/CSS/JavaScript
- localStorage for data persistence
- No backend required
- Mobile responsive design

## Local Development

```bash
# Clone or download the repository
cd bookclub-static

# Open in browser (or use a local server)
open index.html

# Or with Python
python3 -m http.server 8000
```

## Data Storage

All data is stored in your browser's localStorage:
- Reading progress
- Quiz scores
- Notes
- Badge achievements
- Streak counts

**Note:** Data is specific to each browser/device. Clearing browser data will reset progress.
